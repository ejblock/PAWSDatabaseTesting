# PAWSDatabaseTesting
This repository contains quality assurance and unit testing artifacts for a pet tracking and management system developed for a foster care agency. The application is built using C# with Windows Forms (WinForms) for the user interface and integrates with a SQL Server database for persistent data management.

The system supports core operations such as managing pet records, tracking foster placements, and updating animal status through a client-server architecture. The client application handles user interaction and communicates with backend components responsible for business logic and database operations.

This repository specifically highlights the testing and validation layer of the project. Unit tests were implemented using MSTest to verify critical functionality, including data validation, CRUD operations, and client-server communication. Emphasis was placed on ensuring reliability, correctness, and proper error handling across components.

Additionally, the included QA documentation (provided as a compressed archive) outlines test plans, test cases, and results, demonstrating a structured approach to software quality assurance. The project also utilizes modular design through class libraries (DLLs) and follows principles such as separation of concerns and event-driven programming.

This work was completed as part of a software development course and demonstrates practical experience in desktop application development, database integration, and formal testing methodologies.
